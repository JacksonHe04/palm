# PALM Lab - 东南大学模式学习与挖掘实验室

PALM（PAttern Learning and Mining）实验室隶属东南大学计算机科学与工程学院，坐落于九龙湖校区计算机楼，是一个专注于数据挖掘与知识发现、机器学习、模式识别及相关领域的基础性、前沿性、创新性研究机构。

本项目是 PALM 实验室的官方招生网站。

## 技术栈

- **框架**: Next.js 16.2.6
- **UI**: React 19.2.4
- **样式**: Tailwind CSS v4
- **图标**: Lucide React
- **语言**: TypeScript

## 快速开始

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 生产构建
pnpm build

# 启动生产服务器
pnpm start

# 代码检查
pnpm lint
```
